multiassayexperiment.io.anndata module#

multiassayexperiment.io.anndata.from_anndata(adata: AnnData, name: str = 'unknown') → MultiAssayExperiment[source]#

Read AnnData objects as a MultiAssayExperiment.

Since AnnData does not contain sample information, sample named "unknown_sample" will be added to col_data. All cells are considered to be extracted from this sample and is reflected in sample_map.

Parameters:

  • data (AnnData) – An AnnData object.

  • name (str, optional) – Name for the experiment. Defaults to “unknown”.

Returns:

An MAE of data.

Return type:

multiassayexperiment.io.interface module#

multiassayexperiment.io.interface.make_mae(experiments: MutableMapping[str, AnnData | SummarizedExperiment]) → MultiAssayExperiment[source]#

Read a dictionary of experiments as an MultiAssayExperiment.

The import naively creates sample mapping, with each experiment considered to be a independent sample. We add a sample to col_data in this pattern - unknown_sample_{experiment_name}. All cells from the same experiment are considered to be from the same sample and is reflected in sample_map.

Additionally, converts AnnData objects to SingleCellExperiment objects.

Parameters:

experiments (MutableMapping[str, Union[AnnData, SummarizedExperiment]]) –

A dictionary of experiments with experiment names as keys and the experiments as values.

Each experiment can be represented as AnnData objects or any subclass of SummarizedExperiment.

Raises:

  • TypeError – If any of the provided objects are not an expected types.

  • TypeError – If experiments is not a dictionary.

Returns:

An MAE from the experiments.

Return type:

MultiAssayExperiment
